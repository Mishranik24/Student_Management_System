# Student Management System REST API

A complete PHP REST API project with JWT Authentication and Bootstrap frontend.

## Features

- JWT Authentication
- Register/Login
- Student CRUD
- Search Students
- Pagination
- Sorting
- User Profile
- Change Password
- Forgot Password
- OTP Verification
- Reset Password
- Responsive Bootstrap UI
- SweetAlert Notifications
- REST API Architecture

## Technologies Used

- PHP
- MySQL
- JWT
- Bootstrap 5
- JavaScript
- SweetAlert2
- Apache (XAMPP)

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/student-management-system-rest-api.git
```

2. Import the SQL database.

3. Update the database configuration.

4. Start Apache and MySQL.

5. Open:

```
http://localhost/Student_Management_API
```
